# Книги и статьи

## Книги
*   **[glTF](./literature/glTF-2.0.pdf)**: Спецификация формата glTF 2.0

## Статьи
*   **[Shape, Light, and Material Decomposition from Images using Monte Carlo Rendering and Denoising](./literature/Shape_Light_and_Material_Decomposition.pdf)**: Исследование по разделению формы, материалов и освещения из изображений с использованием дифференцируемого рендеринга и денизинга
*   **[MARS: Multi-sample Allocation through Russian roulette and Splitting](./literature/Multi-sample_Allocation_through_Russian_roulette_and_Splitting.pdf)**: Работа по оптимальному распределению сэмплов в многозначной важностной выборке с использованием русской рулетки и расщепления
*   **[ReSTIR BDPT: Bidirectional ReSTIR Path Tracing with Caustics](./literature/ReSTIR_BDPT_Bidirectional_ReSTIR_Path_Tracing_with_Caustics.pdf)**: Исследование по объединению двунаправленной трассировки путей с ReSTIR для эффективного рендеринга каустик
*   **[Real-time Neural Radiance Caching for Path Tracing](./literature/Real-time_Neural_Radiance_Caching_for_Path_Tracing.pdf)**: Метод кэширования радиации с использованием нейронных сетей в реальном времени для ускорения глобального освещения в трассировке путей
*   **[Real-Time 4K Super-Resolution of Compressed AVIF Images. AIS 2024 Challenge Survey](./literature/Real-Time4KSuper-Resolution.pdf)**: Обзор методов реального времени для суперразрешения сжатых AVIF-изображений до 4K
*   **[GPU-Optimized Bi-Directional Path Tracing](./literature/GPU_Optimized_Bi-Directional_Path_Tracing.pdf)**: Оптимизированная для GPU модификация двунаправленной трассировки путей с ограниченным использованием памяти

# Внешние ресурсы и ассеты

## Tutorial
*   **[Tutorial Vulkan 1.3](https://github.khronos.org/Vulkan-Site/tutorial/latest/)**: Урок по Khronos Vulkan 1.3

## Ресурсы
*   **[Intel® Open Image Denoise](https://www.openimagedenoise.org/)**: это открытая библиотека высокопроизводительных, качественных фильтров для снятия шума для изображений, выполненных с помощью трассировки лучей.
